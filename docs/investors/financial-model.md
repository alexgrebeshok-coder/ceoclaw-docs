# Financial Model

## ROI Calculation

Based on MBA thesis research (RANEPA, 2025) and Severavtodor implementation analysis.

### Assumptions

| Parameter | Value |
|-----------|-------|
| Company size | 50-200 employees |
| Number of projects | 5-15 concurrent |
| Annual project portfolio | ₽500M - ₽3B |
| Current PM overhead | 3-5 PMs |
| Average PM salary | ₽180K/month |

### Cost Savings Breakdown

| Category | Annual Savings | Method |
|----------|---------------|--------|
| Reporting automation | ₽36M | 94% reduction in PM reporting time (8hr → 30min/week) |
| Early problem detection | ₽48M | 80% reduction in budget overruns (25% → 5%) |
| Resource optimization | ₽24M | Better allocation across projects |
| Knowledge retention | ₽12M | Reduced rework from lessons learned |
| **Total** | **₽120M/year** | |
