# Road Construction — Severavtodor Case Study

## Context

**Company:** Severavtodor (ООО «Северавтодор»)
**Location:** Yamalo-Nenets Autonomous Okrug (YNAO), Russia
**Industry:** Road and infrastructure construction
**Scale:** 7+ concurrent projects

## The Challenge

Managing road construction in the Arctic means:

- **Short construction season** — 2-3 months of thaw per year
- **Winter roads (зимники)** — supply routes exist only on frozen rivers/tundra
- **Multiple subcontractors** — 10-20 per project, coordination nightmare
- **Remote sites** — hundreds of kilometers from the nearest city
- **Harsh conditions** — temperatures to -50°C, equipment failures common
- **Regulatory complexity** — SNiP, GOST, ФГИС ЦС compliance

## Key Features Used

- **Portfolio Dashboard** — C-level overview of all 7+ projects
- **EVM Analytics** — CPI, SPI tracking per project and portfolio-wide
- **Risk Matrix** — AI-identified risks with mitigation strategies
- **Gantt Charts** — Critical path scheduling with season constraints
- **1C Integration** — Automatic data pull from 1C:ERP

## Lessons Learned

1. **Start with visibility** — Before AI, just having a unified dashboard saves enormous time
2. **EVM is powerful but underused** — Most Russian construction companies don't use EVM at all
3. **AI predictions improve with data** — The more projects tracked, the better the predictions
4. **Mobile is critical** — Foremen and site engineers need mobile-first access
5. **Offline mode is not optional** — Arctic sites regularly lose connectivity
