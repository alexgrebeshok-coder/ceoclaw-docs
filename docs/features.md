# Features

## Current Features (v0.1.0)

### 📊 Dashboard
- Portfolio overview with key KPIs
- AI-generated insights and recommendations
- Project status summary with traffic light indicators
- Budget and schedule health at a glance

### 📋 Project Management
- Project cards with status, budget, timeline, team
- Gantt charts with AI-powered scheduling
- Kanban board with drag-and-drop task management
- Calendar view with milestones and deadlines
- Task hierarchy (epic → task → subtask)

### 🤖 AI Chat
- Natural language queries about project data
- Multi-provider AI (OpenRouter, Zhipu AI, OpenAI)
- Streaming responses for real-time interaction
- Context-aware: understands your project portfolio

### 📈 Analytics & EVM
- Earned Value Management: CPI, SPI, EAC, TCPI
- Trend analysis and variance tracking
- AI-powered delay and cost overrun prediction
- Custom reports and dashboards

### ⚠️ Risk Management
- Risk matrix (probability × impact)
- AI-generated mitigation strategies
- Risk tracking and escalation rules
- Historical risk analysis across projects

### 🗺️ Map Integration
- Yandex Maps integration for project locations
- Geospatial project overview
- Site photos with geotags

### 🌍 Multi-language
- Full localization: Russian, English, Chinese
- User-selectable language preference
- AI responses in selected language

### 🎯 Goals & OKRs
- Strategic goal tracking
- OKR alignment across projects
- Progress visualization

### 👥 Team Management
- Team member profiles and roles
- Resource allocation across projects
- Workload analysis

## In Development

### AI Agents (v0.2)
Five specialized AI agents that work together:

| Agent | Role |
|-------|------|
| **PMO Director** | Strategic decisions, risk assessment, portfolio optimization |
| **Planning Agent** | Auto-scheduling, critical path analysis, resource optimization |
| **Monitoring Agent** | Real-time EVM tracking, deviation alerts, predictions |
| **Financial Agent** | Budget control, cash flow, variance analysis |
| **Knowledge Agent** | RAG-based search, lessons learned, best practices |

### 1C Integration
- Read-only pull from 1C:ERP/PM
- Projects, tasks, resources, financial data
- Bi-directional sync (statuses back to 1C)
- OData / HTTP API

### Voice Interface
- Voice commands for field workers
- Speech-to-text for site reports
- Text-to-speech for dashboard reading

### Offline Mode
- Full functionality without internet
- Sync when connection restored
- Designed for Arctic and remote sites
