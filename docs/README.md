## 🦀 About CEOClaw

CEOClaw is an AI-powered project management platform built specifically for construction companies operating in challenging environments — from Arctic construction sites to remote mining operations.

**Key facts:**
- Built by a construction PM with 24 years of field experience
- 171% projected ROI, ₽120M annual savings per company
- 5 specialized AI agents for automated PMO
- 1C:ERP/PM integration for Russian enterprises
- Free forever for individual project managers
